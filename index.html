Gist

I am using Gist for development of code to test various random bits and pieces. So far there is some Python code and some RFO Basic! snippets on Gist. The Python code has been my most substantial use of git so far. Since I am the only person working on the code I am only using the Master branch. The workflow for the single file of python code on my home PC is very simple -
edit & test code, git add code, git commit, git push and then repeat.
Some of the time I edit in the web browser, then do a git pull to test the code.
